# Compose Specification
- Standardized definition
    - Unify Compose file behavior
        - Supported keys
            - `services`
            - `volumes`
            - `networks`
            - etc.
        - Valid
            - Syntax
            - Semantics
- Owned by
    - Open Container Initiative (OCI)
- Platform-agnostic
    - Not tied to Docker
    - Podman uses it
- Maintained on Github
 - Evolves with support
    - Community
    - Vendors
