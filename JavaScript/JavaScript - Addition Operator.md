# Addition Operator
## Overview
- The addition operator (`+`) adds numbers or concatenates strings.
- Behavior depends on the operand types
## Example
```javascript

// Numeric addition
let x = 5;
let y = 2;
let z = x + y;

// String concatenation
let name = "Jim";

 // "Hello, Jim"
let greeting = "Hello, " + name;
```
