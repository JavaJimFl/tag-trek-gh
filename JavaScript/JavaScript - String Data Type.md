## Notes
- Strings are zero-based character arrays in behavior (e.g., `str[0]`).
- JavaScript automatically converts numbers to strings when using the `+` for concatenation.

## `typeof` Usage
Use the `typeof` operator to check the type of a string.
```javascript
typeof "Hello"; // "string"
```
