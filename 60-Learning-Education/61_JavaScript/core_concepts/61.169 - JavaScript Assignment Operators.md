## Destructuring Assignment

- A special form/subtype of assignment
- Assigns values from arrays or objects to variables
- Done in a single expression
- `const [a, b] = [1, 2];`
- `const {name, age} = person;`

## Chained Assignment

- Assign single value to multiple variables

```javascript
let a, b, c;
a = b = c = 5;
```
