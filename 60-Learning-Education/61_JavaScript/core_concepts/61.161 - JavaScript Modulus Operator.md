# Modulus Operator

## Overview

- The modulus operator (`%`) returns the remainder of division.
- Useful for checking even/odd, cycling through values, etc.

## Examples:

```javascript
let x = 10;
let y = 3;
let remainder = x % y; // 1

// Even or odd
let num = 7;
let isEven = num % 2 === 0; // false
```
